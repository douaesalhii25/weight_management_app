##  Project Description
This project is a simple weight management web application that allows users to log in, record their weight, and track their progress over time. The application works offline using browser storage and provides a basic visualization of data.

---

##  Features
- User login with input validation  
- Add and track daily weight  
- Data stored locally using IndexedDB  
- Chart visualization using HTML Canvas  
- Responsive design using Tailwind CSS and custom CSS  

---

##  Technologies Used
- HTML  
- CSS (Tailwind + Custom CSS)  
- JavaScript  
- IndexedDB  
- Canvas API  

---

##  How to Run
1. Download or clone the project  
2. Open the folder in Visual Studio Code  
3. Run `login.html` using Live Server  
4. Log in and start tracking your weight  

---

##  Learning Outcomes
- Understanding client-side storage (IndexedDB)  
- Implementing form validation in JavaScript  
- Creating dynamic UI updates  
- Working with Canvas for data visualization
